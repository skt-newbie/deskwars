## Game Mechanics

### QR Hunt Rules
1. **5 Attempts per Riddle**: Users get 5 chances to answer each riddle
2. **Case-Insensitive**: Answers are not case-sensitive
3. **Guaranteed Prize**: Correct answer on any attempt wins the guaranteed prize
4. **Next Clue**: After 5 attempts (correct or not), user gets the next clue
5. **Sequential Play**: Must solve riddles in order (QRP_01 → QRP_02 → ... → QRP_15)

### Mystery Gift Rules
1. **One-Time Scan**: Each user can scan each MG code only once
2. **Instant Reward**: Prize is revealed immediately upon scanning
3. **No Riddles**: No questions asked, just scan and claim
4. **Random Distribution**: Prizes are pre-assigned to specific MG codes

### Prize Claiming
- **Physical Prize**: If inventory quantity > 0, user gets physical prize
- **Digital Fallback**: If inventory depleted, user gets equivalent points
- **Auto-Decrement**: Inventory quantity automatically decreases on claim
- **Claim Tracking**: All claims logged in prize_claims table

## Setup Instructions

### 1. Generate QR Codes
Use any QR code generator to create codes for:
- QRP_01 through QRP_15 (15 codes)
- MG_01 through MG_10 (10 codes)

### 2. Print and Laminate
- Print QR codes in a visible size (at least 5cm x 5cm)
- Laminate for durability
- Add decorative elements to make them attractive

### 3. Place QR Codes
- Follow the location mapping above
- Ensure codes are accessible but not too obvious
- Test scanning from different angles and distances

### 4. Test Before Launch
- Scan each QR code with the app
- Verify riddles display correctly
- Test answer validation
- Confirm prize claiming works
