## Notes

- **Real Prizes**: Physical items that will be distributed to winners
- **Dummy Prizes**: When real prizes run out, users receive digital fallback points instead
- **Total Qty**: Total quantity available for each prize item
- **Digital Fallback**: Each item has a point value that's awarded when physical stock is depleted

## Database Integration

This inventory is stored in the `inventory` table with the following schema:
```prisma
model Inventory {
  id                    String @id
  name                  String
  quantity              Int
  digitalFallbackPoints Int    @map("digital_fallback_points")
}
```

The system automatically:
1. Checks if physical stock is available
2. Decrements quantity when claimed
3. Awards digital points when stock reaches zero
